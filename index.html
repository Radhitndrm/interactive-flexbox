Demo Interaktif Flexbox dengan Tailwind CSS

Eksplorasi properti flex-direction, justify-content, align-items, dan align-content secara interaktif dengan drag & drop kotak.

Penjelasan Properti Flexbox

  • flex-direction: arah utama elemen di dalam kontainer (baris / kolom).
  • justify-content: posisi elemen sepanjang sumbu utama.
  • align-items: posisi elemen sepanjang sumbu silang (cross axis).
  • align-content: mengatur jarak antar baris / kolom jika menggunakan flex-wrap.
  • flex-wrap: apakah elemen boleh membungkus ke baris / kolom baru.
  • Seret kotak untuk mengubah urutan (drag and drop).

Properti align-content hanya berfungsi jika ada lebih dari satu baris / kolom (wrap aktif). Jika wrap tidak aktif, properti ini tidak terlihat efeknya.
